# Virtualization Using Cloud Computing

## Overview
This project demonstrates virtualization using AWS EC2, Docker containers, Flask backend APIs, and a frontend dashboard.

## Technologies Used
- AWS EC2
- Docker
- Flask
- React.js
- Python

## Features
- Dockerized virtualization
- Container monitoring
- Flask API integration
- Frontend dashboard

## Team Roles
- Person A: AWS + Backend
- Person B: Frontend + Integration
